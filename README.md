# 📝 Formulário Web - HTML & CSS

![apresentacão do desafio](assets/apresentacao.gif)

Projeto desenvolvido durante minha formação Full Stack, com foco na construção de um formulário completo utilizando **HTML e CSS**, explorando organização modular, estados de interação e layout responsivo.

---

## 🚀 Funcionalidades

- Estrutura completa de formulário com:
  - Inputs de texto
  - Select
  - Date picker customizado
  - Radio buttons personalizados
  - Checkbox customizado
  - Área de upload (dropzone estilizada)
- Validação nativa com `required` e `:invalid`
- Estados visuais para `hover`, `focus` e `checked`
- Layout com CSS Grid
- Responsividade para telas menores
- Organização modular do CSS com `@import`

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- CSS Grid
- Variáveis CSS (`:root`)
- Pseudo-classes modernas (`:has`, `:focus-within`, `:invalid`)
